这些关键词

{
"result": [ ← 搜索结果列表（数组，每个是一个搜索结果）
{
"title": "…", ← 搜索结果标题（有些是Unicode转义）
"link": "…", ← 搜索结果链接（通常是PDF、网页）
"snippet": "…" ← 搜索结果的摘要（也是Unicode转义）
},
…
],
"relatedSearches": [ ← 相关搜索推荐（链接数组）
"https://…",
…
]
}
