    У»итѕєТ«хУДБжЄіС╗ЦУЅ▓тѕЌжЄЇТќ░тЇажбєтіаТ▓ЎуџёУ«Атѕњ№╝їтїЁТІг№╝џ1.тєЏС║ІуЏ«ТаЄсђЂ2.тЁиСйЊУАїтіесђЂ3.С║║жЂЊСИ╗С╣Ѕтй▒тЊЇсђЂ4.тЏйжЎЁтЈЇт║ћСИјтцќС║цтіеТђЂсђЂ5.ТйютюежБјжЎЕСИјТїЉТѕўсђѓ

    СИЅсђЂТўјуА«ТеАтъІУДњУЅ▓СИјС╗╗тіА¶

    уЏ«уџё№╝џ ТїЄт«џТеАтъІтюеућЪТѕљТЌХуџёУДњУЅ▓У║ФС╗й№╝їТюЅтіЕС║јтЁХС╗јуЅ╣т«џУДєУДњУЙЊтЄ║СИЊСИџТђДуГћТАѕсђѓ ТіђТю»тјЪуљє№╝џ

    • ТїЄт«џУДњУЅ▓№╝ѕтдѓТЃЁТіЦтѕєТъљтИѕсђЂтЏйжЎЁт«ЅтЁежЌ«жбўСИЊт«ХсђЂтю░у╝ўТћ┐Т▓╗тГдУђЁ№╝ЅтЈ»С╗ЦУ«ЕТеАтъІтюеућЪТѕљТЌХУЄфтіежЄЄућеУ»ЦжбєтЪЪуџёУ»ГУеђжБјТа╝тњїСИЊСИџуЪЦУ»єсђѓ
    • жђџУ┐Є**ТїЄС╗цТЈљуц║У»Ї**№╝їУ«ЕТеАтъІтюеТїЄт«џТЃЁтбЃСИІућЪТѕљуГћТАѕ№╝їТЈљжФўтєЁт«╣уџё**СИЊСИџТђДСИјтЈ»С┐Ат║д**сђѓ

    ућеТѕиУЙЊтЁЦУдЂуѓ╣№╝џ

    • ТїЄт«џУ║ФС╗й№╝џТЃЁТіЦтѕєТъљтИѕсђЂтЏйжЎЁт«ЅтЁежЌ«жбўСИЊт«ХсђЂтю░у╝ўТћ┐Т▓╗тГдУђЁуГЅ
    • ТїЄт«џС╗╗тіА№╝џТЃЁТіЦУ»ёС╝░сђЂжБјжЎЕжбёТхІсђЂт▒ђті┐УДБУ»╗сђЂУХІті┐т»╣Т»ћ
    • жЎљт«џжБјТа╝№╝џТГБт╝ЈсђЂтГдТю»сђЂТѕўуЋЦТіЦтЉісђЂТќ░жЌ╗у«ђТіЦ

    уц║СЙІ№╝џ¶

    ТЎ«жђџТЈљжЌ«№╝џ

    С╗ЦУЅ▓тѕЌжЄЇТќ░тЇажбєтіаТ▓ЎуџёУ«Атѕњ

    Тќ╣т╝ЈСИђ№╝џСИЊт«ХУ║ФС╗й + тѕєТъљС╗╗тіА¶

    т░єТеАтъІУ«Йт«џСИ║**жбєтЪЪСИЊт«Х**№╝їУ«ЕтЁХС╗јСИЊСИџУДњт║дУ┐ЏУАїтѕєТъљсђѓ

    жЌ«жбўТћ╣тєЎ№╝џ

    тЂЄУ«ЙСйаТў»СИђтљЇтЏйжЎЁт«ЅтЁежЌ«жбўСИЊт«Х№╝їУ»иС╗јС╗ЦСИІу╗┤т║дтѕєТъљС╗ЦУЅ▓тѕЌжЄЇТќ░тЇажбєтіаТ▓ЎуџёУ«Атѕњ№╝џ
 1. ТѕўуЋЦуЏ«ТаЄ№╝џС╗ЦУЅ▓тѕЌуџётєЏС║ІтњїТћ┐Т▓╗ТёЈтЏЙсђѓ
 2. тЁиСйЊУАїтіе№╝џС╗ЦУЅ▓тѕЌтЏйжў▓тєЏ№╝ѕIDF№╝ЅтЈ»УЃйжЄЄтЈќуџёТѕўТю»СИју«АуљєТјфТќйсђѓ
 3. тЏйжЎЁтй▒тЊЇ№╝џуЙјтЏйсђЂУЂћтљѕтЏйуГЅтЏйт«ХСИју╗ёу╗ЄуџёуФІтю║СИјт║ћт»╣ТјфТќйсђѓ
 4. ТйютюежБјжЎЕ№╝џС╗ЦУЅ▓тѕЌтЈ»УЃйжЮбСИ┤уџёжЋ┐ТюЪтЈЏС╣▒сђЂтцќС║цтјІтіЏтњїС║║жЂЊСИ╗С╣ЅтЇ▒Тю║сђѓ  
     У»ит╝ЋућеУ┐ЉТюЪтЏйжЎЁтЁ│у│╗ТіЦтЉіТѕќтјєтЈ▓ТАѕСЙІУ┐ЏУАїУ«║У»Ђсђѓ

    Тќ╣т╝ЈСИЅ№╝џТЋ░ТЇ«тѕєТъљтИѕ + жбёТхІС╗╗тіА¶

    ТїЄт«џТеАтъІСИ║**ТЋ░ТЇ«тѕєТъљтИѕ**№╝їУдЂТ▒ѓУ┐ЏУАї**ТЋ░ТЇ«жЕ▒тіеуџёжбёТхІ**сђѓ

    жЌ«жбўТћ╣тєЎ№╝џ

     тЂЄУ«ЙСйаТў»СИђтљЇТЋ░ТЇ«тѕєТъљтИѕ№╝їУ»итЪ║С║јС╗ЦСИІТЋ░ТЇ«у╗┤т║д№╝їжбёТхІС╗ЦУЅ▓тѕЌжЄЇТќ░тЇажбєтіаТ▓ЎтЈ»УЃйтИдТЮЦуџётй▒тЊЇ№╝џ
 1. тєЏС║ІУАїтіеТЋ░ТЇ«№╝џУ┐ЉТюЪС╗ЦУЅ▓тѕЌтєЏС║ІУАїтіеТгАТЋ░СИјУДёТеАсђѓ
 2. С║║жЂЊСИ╗С╣Ѕу╗ЪУ«А№╝џтіаТ▓Ўтю░тї║жџЙТ░ЉС║║ТЋ░СИјС║║жЂЊСИ╗С╣ЅТЈ┤тіЕТЃЁтєхсђѓ
 3. тЏйжЎЁтИѓтю║тЈЇт║ћ№╝џСИГСИютє▓уфЂт»╣уЪ│Т▓╣С╗иТа╝СИјжЄЉУъЇтИѓтю║уџётй▒тЊЇсђѓ  
    У»иСй┐уће**ТЋ░ТЇ«УАеТа╝ТѕќтЏЙУАе**т▒Ћуц║жбёТхІу╗ЊТъю№╝їт╣Хтюеу╗ЊУ«║жЃетѕєУ┐ЏУАїтѕєТъљТђ╗у╗Њсђѓ

    С║ћсђЂТїЄт«џУЙЊтЄ║у▓њт║д¶

    уЏ«уџё№╝џ ТјДтѕХућЪТѕљтєЁт«╣уџёУ»ду╗єуеІт║д№╝їТ╗АУХ│тЁиСйЊжюђТ▒ѓсђѓ ТіђТю»тјЪуљє№╝џ

    • У»ГУеђТеАтъІтюеућЪТѕљтєЁт«╣ТЌХ№╝їтдѓТъюућеТѕиТюфТїЄт«џУЙЊтЄ║у▓њт║д№╝їтЈ»УЃйућЪТѕљУ┐ЄС║јт«йТ│ЏТѕќуг╝у╗ЪуџётєЁт«╣сђѓ
    • жђџУ┐ЄУ«Йт«џ**у▓њт║дТѕќу╗┤т║д**№╝їТеАтъІтЈ»С╗ЦућЪТѕљТЏ┤УЂџуёдСИћСИ░т»їуџёуГћТАѕсђѓ

    ућеТѕиУЙЊтЁЦУдЂуѓ╣№╝џ

    • ТїЄт«џУЙЊтЄ║у▓њт║д№╝ѕТдѓТІг/У»ду╗є№╝Ѕ
    • У«Йт«џтЁиСйЊу╗┤т║дТѕќт▒ѓТгА
    • жЎљт«џтГЌТЋ░Тѕќу»Єт╣Ё

    уц║СЙІ№╝џ¶

    • т«йТ│ЏТЈљжЌ«№╝џ РђюућЪТѕљтЁ│С║јAIуЏЉу«АТћ┐уГќуџётєЁт«╣сђѓРђЮ
    • у▓Йу╗єтїќТЈљжЌ«№╝џ РђюУ»итѕєСИЅСИфт▒ѓТгА№╝ѕТћ┐т║юсђЂС╝ЂСИџсђЂтЁгС╝Ќ№╝ЅУ»ду╗єтѕєТъљAIуЏЉу«АТћ┐уГќуџётй▒тЊЇсђѓТ»ЈСИфт▒ѓТгАућЪТѕљСИЇт░ЉС║ј300тГЌ№╝їт╣Хт╝ЋућеуЏИтЁ│ТАѕСЙІсђѓРђЮ

    Тќ╣т╝ЈСИђ№╝џтѕєт▒ѓТгАУЙЊтЄ║ + ТїЄт«џтГЌТЋ░¶

    т░єжЌ«жбўТІєУДБСИ║**СИЇтљїу╗┤т║дТѕќт▒ѓТгА**№╝їт╣ХТїЄт«џ**тГЌТЋ░УдЂТ▒ѓ**№╝їУ«ЕУЙЊтЄ║ТЏ┤ТюЅТЮАуљєтњїТи▒т║дсђѓ

    жЌ«жбўТћ╣тєЎ№╝џ

    У»итѕєСИЅСИфт▒ѓТгАУ»ду╗єтѕєТъљС╗ЦУЅ▓тѕЌжЄЇТќ░тЇажбєтіаТ▓Ўуџётй▒тЊЇ№╝џ
тєЏС║Іт▒ѓжЮб№╝џС╗ЦУЅ▓тѕЌтєЏжўЪуџётЁиСйЊУАїтіеуГќуЋЦсђЂуЏ«ТаЄСИјтљјтІцжЃеуй▓сђѓ
Тћ┐Т▓╗т▒ѓжЮб№╝џС╗ЦУЅ▓тѕЌСИјти┤тІњТќ»тЮдуџётцќС║цт▒ђті┐тЈўтїќ№╝їтЏйжЎЁуцЙС╝џуџёТђЂт║дсђѓ
С║║жЂЊСИ╗С╣Ѕт▒ѓжЮб№╝џт»╣тіаТ▓Ўт╣│Т░Љуџётй▒тЊЇ№╝їтїЁТІгжџЙТ░ЉТй«сђЂуЅЕУхёуЪГу╝║тњїС║║жЂЊтЇ▒Тю║сђѓ
Т»ЈСИфт▒ѓТгАСИЇт░ЉС║ј300тГЌ№╝їт╣Хт╝ЋућеуЏИтЁ│ТАѕСЙІТѕќТЋ░ТЇ«У┐ЏУАїУ«║У»Ђсђѓ

    Тќ╣т╝ЈС║ї№╝џТїЄт«џУЙЊтЄ║у▓њт║д№╝ѕТдѓТІг vs. У»ду╗є№╝Ѕ¶

    ТўјуА«УдЂТ▒ѓ**тЁѕТдѓТІгтљјУ»ду╗є**№╝їУ«ЕТеАтъІтЁѕу╗ЎтЄ║у«ђУдЂТдѓУДѕ№╝їтєЇУ┐ЏУАїТи▒тЁЦтѕєТъљсђѓ

    жЌ«жбўТћ╣тєЎ№╝џ

    У»иТїЅС╗ЦСИІСИцжЃетѕєУЙЊтЄ║С╗ЦУЅ▓тѕЌжЄЇТќ░тЇажбєтіаТ▓Ўуџётй▒тЊЇ№╝џ
1№ИЈ. ТдѓТІгжЃетѕє№╝ѕ150тГЌтєЁ№╝Ѕ№╝џућеу«ђТ┤ЂУ»ГУеђТђ╗у╗ЊТЋ┤СйЊтй▒тЊЇ№╝їтїЁТІгтєЏС║ІсђЂТћ┐Т▓╗тњїС║║жЂЊСИ╗С╣Ѕт▒ѓжЮбсђѓ
2. У»ду╗єжЃетѕє№╝ѕТ»ЈжЃетѕє300тГЌ№╝Ѕ№╝џтѕєтѕФС╗јС╗ЦСИІСИЅСИфу╗┤т║дУ┐ЏУАїУ»ду╗єтѕєТъљ№╝џ
тєЏС║І№╝џтЁиСйЊТѕўТю»СИјтЈ»УЃйТїЂу╗ГТЌХжЌ┤
Тћ┐Т▓╗№╝џС╗ЦУЅ▓тѕЌСИјти┤тІњТќ»тЮдтЁ│у│╗уџётЈўтїќ
С║║жЂЊСИ╗С╣Ѕ№╝џтіаТ▓Ўт╣│Т░ЉуіХтєхСИјтЏйжЎЁТЈ┤тіЕтЈЇт║ћ

    тЁГсђЂтљ»ућетцџУй«У┐йжЌ«¶

    уЏ«уџё№╝џ жђџУ┐ЄтцџУй«т»╣У»Ю№╝їжђљТГЦт╝Ћт»╝ТеАтъІућЪТѕљ**ТЏ┤Ти▒тЁЦТѕќУАЦтЁЁТђД**уџётєЁт«╣сђѓ ТіђТю»тјЪуљє№╝џ

    • тцџУй«т»╣У»Ют░єтцЇТЮѓжЌ«жбўТІєУДБСИ║тцџСИфт░ЈжЌ«жбў№╝їТюЅтіЕС║јТеАтъІућЪТѕљТЏ┤у╗єУЄ┤уџёуГћТАѕсђѓ
    • Т»ЈУй«т»╣У»ЮтЪ║С║јСИіТќЄУ»ГтбЃУ┐ЏУАїућЪТѕљ№╝їТЈљжФў**жђ╗УЙЉУ┐ъУ┤»ТђДСИју╗єУіѓСИ░т»їт║д**сђѓ

    ућеТѕиУЙЊтЁЦУдЂуѓ╣№╝џ

    • тѕЮтДІжЌ«жбў№╝џТЈљтЄ║ТЃЁТіЦТѕќт«ЅтЁеС║ІС╗ХуџёТЋ┤СйЊжЌ«жбў
    • У┐йжЌ«№╝џС╗ју╗єУіѓУДњт║дУАЦтЁЁТЈљжЌ«№╝їтдѓтіетЏасђЂтй▒тЊЇсђЂт║ћт»╣ТјфТќй
    • Ти▒т║дУ┐йжЌ«№╝џТјбУ«етЏаТъюсђЂУХІті┐Тѕќу╗єУіѓти«т╝ѓ№╝їУ┐ЏУАїТЃЁТіЦтѕєТъљТѕќжБјжЎЕжбёТхІ

    уц║СЙІ№╝џ¶

    • ућеТѕи№╝џРђю2024т╣┤С║џтцфтю░тї║уџёуйЉу╗ют«ЅтЁетеЂУЃЂУХІті┐Тў»С╗ђС╣ѕ№╝ЪРђЮ
    • ТеАтъІтЏъуГћтљј№╝їућеТѕиУ┐йжЌ«№╝џ
      • тЊфС║ЏтЏйт«ХТѕќтю░тї║Тюђт«╣ТўЊТѕљСИ║Тћ╗тЄ╗уЏ«ТаЄ№╝Ъ
      • тйЊтЅЇСИ╗УдЂТћ╗тЄ╗ТЅІТ│ЋТюЅтЊфС║Џ№╝Ъ
      • ТюфТЮЦСИцт╣┤уйЉу╗ют«ЅтЁетеЂУЃЂТў»тљдС╝џУ┐ЏСИђТГЦтіатЅД№╝Ъ
